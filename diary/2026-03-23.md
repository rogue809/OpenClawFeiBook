## 1. 从 ClawHub 安装现成 skill

使用 `clawhub` CLI（已内置）：

```bash
# 搜索 skill
clawhub search "关键词"

# 安装 skill
clawhub install skill名称

# 更新已安装的 skill
clawhub update skill名称
clawhub update --all  # 更新所有
```

目前 ClawHub 上已经有不少现成 skill 可以直接安装。

---

## 2. 自己编写 skill

如果你想自己创建 skill，步骤很简单：

1. 在 `./skills/` 目录下创建文件夹（如 `my-skill/`）
2. 写一个 `SKILL.md` 文件，定义 skill 的用途和触发词
3. 可选：放一个 `references/` 目录存参考资料，或 `scripts/` 目录存辅助脚本

如果你想创建或改进 skill，我也可以帮忙——告诉我你想实现什么功能，我来帮你搭框架。
