# 详细日记 (Detailed Diary)

将你和我的聊天记录整理成 Markdown 日记文档。

## 触发方式

当我对你说以下内容时，自动触发：
- "写日记"
- "生成日记"
- "记录今天聊天"
- "把聊天记下来"

## 使用方法

### 1. 基本用法

```
你：写日记
我：请问要记录哪段时间的聊天？（例如：今天早上、昨天、2026-03-23）
```

### 2. 指定时间

```
你：写昨天的日记
我：（获取昨天的聊天记录，生成文档）
```

### 3. 生成文档

- 文件名格式：`diary/YYYY-MM-DD.md`
- 内容包含：
  - 日期时间
  - 参与者标记（你 / 麻辣小龙虾）
  - 聊天内容（按时间顺序）
  - 可选：生成摘要

### 4. Git 推送

当你说"上传日记"或"推送日记"时：
- 自动将新增/修改的日记文件加入 git
- 推送到远程仓库

## 输出示例

```markdown
# 2026-03-23 日记

## 时间范围
- 开始：2026-03-23 10:00
- 结束：2026-03-23 12:00

## 参与者
- 邱顺（用户）
- 麻辣小龙虾（助手）

## 聊天记录

### 10:00 邱顺
今天天气怎么样？

### 10:01 麻辣小龙虾
今天是晴天，气温 20-28°C。

### 10:05 邱顺
帮我查一下昨天我们讨论的那个文档。

### 10:06 麻辣小龙虾
好的，我查一下...

---

## 摘要
（可选：自动生成对话摘要）
```

## 文件存储

- 路径：`/home/gem/workspace/agent/workspace/diary/`
- 命名：`YYYY-MM-DD.md`

## Git 操作

- 手动触发（需要你明确说"上传"）
- 会把 diary/ 目录下的所有文件一起推送

## 注意事项

- 仅记录当前会话的聊天记录
- 不涉及其他群聊或私聊
- 尊重隐私，不记录敏感信息
